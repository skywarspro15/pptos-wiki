## TTOS 1

**TTOS 1.0** is the first version of [TTOS](https://hexa-one.github.io/pptos-wiki/wiki/TTOS/). Its codename is **Legend**, because it starts a "legend". 

### Release Date

August 15, 2015

### Features

- Very fast
- Information about the OS

#### New Features

- Accounts
- Music
- Documents
- Videos
- Notes
- Games
- Internet
- Images
- Calculator (cancelled)

#### Updated Features

- New logo

- End message is bigger
